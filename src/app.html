 Add blocks  to your model. After you open StatSim or create a new model, add some elements (blocks) to it. There are multiple types of blocks. Data are typically constants or input parameters, expressions set relationships between blocks, random variables describe stochastic values.

Update blocks settings. Each block has its own settings. They determine how the program will evaluate the block and how it will be used later. Almost all elements have the "Show in results" checkboxes. Such checkbox determines if the block will be shown in output.

Set simulation Params. After you created and set all needed blocks it's time to choose a simulation method. The simple rule is if you don't use random variables select Deterministic, otherwise choose Rejection Sampling or MCMC.

 Run  your model, but save your project first! If you don't use a server for simulations, StatSim can become unresponsive. If everything is ok, you will see results as beautiful charts or scalar values. Compile your model to binaries if you want to use it as a separate app.
